# trabalho_de_web ChatWeb FlowChat

Um sistema de chat web utilizando firebase.

---

## Tecnologias Utilizadas

- **Frontend:** HTML, CSS, JavaScript  
- **Backend / Database:** Firebase Realtime Database  
- **Controle de versão:** Git e GitHub  
- **Ferramentas auxiliares:** GitHub Desktop, VSCode

---

## Funcionalidades

- Envio e recebimento de mensagens públicas e privadas 
- Layout adaptável a telas de desktop e mobile  
- Histórico de mensagens armazenado no Firebase
- Sistema de autenticação com Firebase (Google)
- Sistema de presença utilizando Firebase

---

## Instalação e Uso
1. Clone o repositório:
   ```bash
   git clone https://github.com/vitorHBF12/trabalho_de_web.git
   cd trabalho_de_web
2. Abra o projeto no navegador ou use um servidor local (ex.: Live Server no VSCode)
3. Configure seu Firebase no arquivo de configuração (firebaseConfig.js) com suas credenciais
4. Acesse a aplicação e teste as mensagens públicas e privadas
